# Iconography

---

Fractal uses the [Unicons icon library](https://iconscout.com/unicons) in the
[Line version (free)](https://iconscout.com/unicons/explore/line).

## Usage

The `@iconscout/react-unicons` package is available as a Fractal dependency so
you can import any Line icon in your React components and use it as any other
React component:

```jsx
import { UilIcons } from '@iconscout/react-unicons'
```

A good practice is to rename the imported icon in the `<Name>Icon` format, for
example:

```jsx
import { UilHeart as HeartIcon } from '@iconscout/react-unicons'
```
